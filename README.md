# qr-image

Create a QR image from input text.

NodeJs application written with
* Node JS framework
* qr-image.

## Usage:
node run index.js

* Enter your text to encode.
  * e.g. www.google.com
* Select an output format.
  * e.g. png
* Select an output filename.
  * e.g google.png

File will br created in root folder.
